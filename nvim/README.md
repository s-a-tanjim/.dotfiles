# Dependencies
- Lazy: Plugin manager

# Plugins
- telescope: File finder
- treesitter: Syntax highlighting
- lsp-zero: LSP server
- tokyonight: ColorScheme
- nvim-tree: File tree in side menu
- bufferline: Tabs


# Create a symlink
```bash
ln -s ~/<base-path>/dotfiles/nvim ~/.config/nvim
```
